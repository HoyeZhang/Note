# 路径之和
> 给定一个二叉树与整数sum，找出所有从根节点到叶节点的路径，这些路径上的节点值为sum
## 解题思路
- 遍历二叉树的思想，结合栈的使用
## 代码
<pre>
void EqualSum(BiTree T,Stack &S,int sum,int &x)
{
    if(T==NULL)
    {
        return;
    }
    Push(S,T->data);
    if(T->lchild==NULL && T->rchild==NULL && S.sum==sum)
    {
        PrintStack(S);
    }
    EqualSum(T->lchild,S,sum,x);
    EqualSum(T->rchild,S,sum,x);
    Pop(S,x);
}

</pre>
